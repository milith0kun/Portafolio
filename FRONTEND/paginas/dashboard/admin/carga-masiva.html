Carga Masiva de Datos

Utilice esta herramienta para cargar datos masivamente al sistema.

Requerido para carga de asignaturas y asignaciones.
Carga de Usuarios

Cargue un archivo Excel con la lista de usuarios a crear en el sistema.

El archivo debe contener las columnas: nombre, apellidos, email, dni, rol (administrador, docente, verificador).
Carga de Asignaturas

Cargue un archivo Excel con la lista de asignaturas a crear en el sistema.

El archivo debe contener las columnas: codigo, nombre, carrera, semestre, creditos, tipo (obligatorio, electivo).
Carga de Asignaciones Docente-Asignatura

Cargue un archivo Excel con las asignaciones de docentes a asignaturas.

El archivo debe contener las columnas: codigo_asignatura, email_docente.
Inicialización Completa del Sistema

Los archivos requeridos son:

  1. Ciclos académicos
  2. Roles
  3. Usuarios
  4. Carreras
  5. Asignaturas
  6. Asignaciones docente-asignatura
  7. Criterios de evaluación
