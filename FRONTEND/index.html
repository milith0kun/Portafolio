Flujo del Proceso Académico

Conoce cómo funciona el sistema paso a paso

1. Configuración Inicial

Administradores crean el ciclo académico, definen asignaturas y asignan docentes a materias.

Crear ciclo académico Asignar docentes a asignaturas Asignar verificadores

2. Apertura del Período

Administradores abren el período de subida de documentos para que los docentes puedan trabajar.

Activar subida de documentos Notificar a docentes Habilitar portafolios

3. Trabajo Docente

Docentes suben sus documentos académicos, completan portafolios y monitorean su progreso.

Subir documentos Completar portafolios Ver progreso

4. Cierre de Subidas

Administradores cierran el período de subidas y activan la fase de verificación.

Cerrar período de subidas Activar verificación Notificar verificadores

5. Proceso de Verificación

Verificadores revisan documentos, crean observaciones y aprueban o rechazan contenido.

Revisar documentos Crear observaciones Aprobar/rechazar

6. Reportes y Cierre

Administradores generan reportes finales, analizan estadísticas y cierran el ciclo académico.

Generar reportes Análisis estadístico Cerrar ciclo

Acceso por Roles

Seleccione su rol para conocer las funcionalidades disponibles

Administrador

Control Total

Responsabilidades:

  • Configurar ciclos académicos
  • Gestionar usuarios y roles
  • Asignar docentes a asignaturas
  • Controlar estados del sistema
  • Generar reportes ejecutivos
  • Cargar datos masivos (Excel)
Acceder como Administrador

Docente

Gestión de Portafolios

Responsabilidades:

  • Gestionar portafolios académicos
  • Subir documentos requeridos
  • Monitorear progreso de completitud
  • Revisar observaciones recibidas
  • Responder a verificadores
  • Generar reportes personales
Acceder como Docente

Verificador

Control de Calidad

Responsabilidades:

  • Revisar documentos académicos
  • Crear observaciones detalladas
  • Aprobar o rechazar contenido
  • Gestionar cola de revisión
  • Comunicarse con docentes
  • Generar estadísticas de verificación
Acceder como Verificador
